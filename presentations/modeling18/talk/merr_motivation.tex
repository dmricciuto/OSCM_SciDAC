\vspace*{2mm}
\only<1>{
\bi \setlength{\leftmargini}{-2pt}
\item Given noisy data -- Gaussian noise  
\item $y=g_\mathrm{true}(x)+\epsilon$
\ei
}
\only<2-3>{
\bi \setlength{\leftmargini}{-2pt}
\item Employ Bayesian inference to fit an exponential model: $y_\mathrm{m}=f(x;\lambda)$
\item Discrepancy between data and prediction presumed exclusively due to \emph{i.i.d.} Gaussian data noise: $y=f(x;\lambda)+\epsilon_\mathrm{d}$
\only<2>{
\item Plotted: 
\bi
\item Posterior density on the parameters
\item Preditive mean and standard deviation
\ei
}
\only<3>{
\item True model $g(x)$ -- dashed-red -- differs from fit model $f(x,\lambda)$
\item Actual discrepancy includes both data and model errors
}
\ei
}
\only<4-5>{
\bi \setlength{\leftmargini}{-2pt}
\item Increasing number of data points decreases posterior and predictive uncertainty
\item We are increasingly sure about predictions based on the \emph{wrong} model
\ei
}
\only<6>{
\bi \setlength{\leftmargini}{-2pt}
\item If the model has structural uncertainty, more data leads to biased and overconfident results
\item We want to quantify model-vs-truth discrepancy in a rigorous and systematic way
\bi
\item Cannot ignore model error
\ei
\ei
}
}
